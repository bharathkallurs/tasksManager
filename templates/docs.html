FAQs

What is Task Management Application?
Task Management Application (TMA) is an application to perform CRUD operations on task. It is a simple tool which can also let you list and filter your tasks list.
What is the backend developed on?
TMA backend work is written in python Flask.
What is the frontend developed on?
TMA frontend is written using jinja2 for templating, angular.js for frontend MVC and twitter bootstrap for styling.
What all APIs are allowed at the backend?
API implementation is REST based. Following are allowed:
  • Add task : /task/add 
                                 
                                 {
                                  "task": {
                                    "task_name": "string",
                                    "task_description": "string",
                                    "task_owner": "string",
                                    "task_end_date": "YYYY-MM-DD"
                                  }
                                }
  • Delete task: /task/delete/{task-id}
  • Update task: /task/update/{task-id} 
                                  
                                  {
                                  "task": {
                                    "task_create_date": "string",
                                    "task_description": "string",
                                    "task_end_date": "YYYY-MM-DD",
                                    "task_id": {task-id},
                                    "task_name": "string",
                                    "task_owner": "string"
                                  }
                                }
  • Get task: /task/get/{task-id}
  • List all tasks: /task/list
  • list all tasks before given date: /task/list_before/{YYYY-MM-DD}
  • list all tasks created after a date: /task/list_after/{YYYY-MM-DD}
  • list all tasks created within a date range: /task/list_bw_cr_dt/{YYYY-MM-DD}/{YYYY-MM-DD}
What operations are supported in frontend?
  • Task Create
  • Task Delete at ach task level
  • Task Update at each task level
  • Task List all
  • Task list created within a date range
What are the frontend libraries used for this app?
  • Twitter Bootstrap
  • Angular.js 1.2
  • Angular-sanitize.js
  • Angular-animate.js
  • Angular-datepicker
  • ngToast for notifications
  • Material Design for Bootstrap (MDB)
What is the database used for this application?

The app uses MongoDB instance hosted on heroku

Where is the app hosted?

TMA is hosted on a PaaS platform called heroku
